## 安装

确保先安装依赖：

```bash
# pnpm
pnpm install
```

## 开发

启动开发服务：

```bash
# Vue 2.6.x
pnpm run dev:2
# Vue 2.7.x
pnpm run dev:2.7
# Vue 3
pnpm run dev:3
```

## 构建

构建库用于生产或发布：

```bash
# build all versions
pnpm run build
```
