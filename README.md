## Setup

Make sure to install the dependencies:

```bash
# pnpm
pnpm install
```

## Development Server[](https://)

Start the development server

```bash
# Vue 2.6.x
pnpm run dev:2
# Vue 2.7.x
pnpm run dev:2.7
# Vue 3
pnpm run dev:3
```

## Production

Build the library for production or publish:

```bash
# build all versions
pnpm run build
```
